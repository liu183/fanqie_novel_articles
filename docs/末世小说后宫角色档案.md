## 📊 角色分类总览

### 分类标准

根据角色的重要性、美貌等级、戏份占比和团队价值，将后宫角色分为三类：

1. **核心后宫**：必须有独立戏份，推动剧情发展
2. **重要配角**：有特殊作用，但戏份相对较少
3. **边缘角色**：功能单一，可删减或合并

### 后宫成员统计（第1-86章）

**总人数：10人**
- 核心后宫：5人
- 重要配角：3人
- 边缘角色：2人

- 10个后宫成员，读者记不住
- 前5个角色戏份严重不足

**问题3：缺乏深度**

- 每个角色都缺乏深度刻画
